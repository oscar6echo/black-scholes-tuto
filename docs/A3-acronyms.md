# Acronyms

| Acronym | Full form | Notes |
|---|---|---|
| ATM | At-The-Money | $S \approx K$ |
| BS | Black-Scholes | The option pricing model derived in [Section 1](./01-bs-pde) |
| GBM | Geometric Brownian Motion | $dS = \mu S\,dt + \sigma S\,dW$; log-returns are normally distributed. See [Appendix A1](./A1-ito-lemma) |
| ITM | In-The-Money | Call: $S > K$; put: $S < K$ |
| OTM | Out-of-The-Money | Call: $S < K$; put: $S > K$ |
| PDE | Partial Differential Equation | The BS PDE governs derivative prices under no-arbitrage. See [Section 1](./01-bs-pde) |
| SDE | Stochastic Differential Equation | Equation of the form $dX = a\,dt + b\,dW$ governing a stochastic process |
